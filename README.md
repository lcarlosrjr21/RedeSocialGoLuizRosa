# Rede Social em Go - LuizRosa
Projeto Rede Social com Go de um curso na Udemy

API para cadastrar, atualizar dados, atualizar senha (antiga pra nova), buscar todos usuarios, buscar um usuario, deletar usuario, criar publicações, atualizar publicacoes, buscar publicacao especifica, buscar publicacoes de um determinado usuario, curtir publicaco (por enquanto). Códigos comentados na medida do possível.
